## Superstore Sales Performance Analysis (2014–2017)

# Project Overview 
This project analyses retail sales data to uncover key business insights and support data-driven decision-making. The analysis combines Microsoft Excel for dashboarding and Microsoft SQL Server for data querying and transformation.

# Objectives

The project aims to answer key business questions:

Which products generate the most revenue?
How do sales change over time?
Which categories and regions are most profitable?
How do discounts impact profitability?

# Tools Used
Microsoft Excel (Data Cleaning, PivotTables & Dashboarding)
Microsoft SQL Server (Data Analysis & Queries)

# Dashboard Preview

The dashboard provides an overview of sales performance, including revenue trends, top-performing products, regional analysis, and profitability insights.

# Key Insights
Sales showed consistent growth from 2014 to 2017
Sales peak during November and December, with the highest performance in November 2017
The Technology category generates the highest revenue and profit
High discount levels negatively impact profitability
The West region performs the strongest in both sales and profit

# SQL Analysis

SQL Server was used to perform data analysis and extract key insights, including:

Total revenue and profit calculations
Profit margin analysis
Sales trends over time
Top-performing products
Category and regional performance
Discount impact on profitability

# Conclusion & Recommendations

Based on the analysis:

Focus on the Technology category to drive revenue growth
Leverage peak sales periods (Nov–Dec) for marketing campaigns
Reduce excessive discounting to improve profit margins
Expand strategies in high-performing regions such as the West
Strengthen strategies in underperforming regions
