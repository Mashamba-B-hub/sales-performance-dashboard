# Superstore Sales Performance Analysis (2014–2017)

# Project Overview
This project analyses business sales data to uncover trends, top-performing products, and profitability insights. The goal is to support data-driven decision-making and identify areas for business growth.

# Tools Used
- Microsoft Excel  
- Pivot Tables  
- Data Cleaning  
- Data Visualisation  

# Key Insights
- Sales grew steadily from 2014 to 2017, with peaks in November and December  
- The highest sales were recorded in November 2017  
- Technology is the top-performing category in both revenue and profit  
- High discounts negatively impact profitability  
- The West region shows the strongest overall performance  

# Recommendations
- Reduce excessive discounting to improve profit margins  
- Focus on high-performing categories such as Technology  
- Strengthen strategies in underperforming regions  
- Leverage seasonal trends by increasing inventory and marketing during peak periods
